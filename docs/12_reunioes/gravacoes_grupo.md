# Gravações da Equipe

## Introdução

Esta página apresenta as gravações produzidas pela equipe ao longo das entregas da disciplina.

Os vídeos documentam a evolução do projeto durante cada unidade, incluindo apresentações, discussões técnicas, organização das atividades e demonstração dos artefatos desenvolvidos pela equipe.

Esses registros contribuem para acompanhar o progresso do trabalho e evidenciam a aplicação dos conceitos de Engenharia de Requisitos durante o desenvolvimento da solução.
