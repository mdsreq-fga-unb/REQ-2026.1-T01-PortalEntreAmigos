# [US08](mvp.md)
> **Como moderador, quero excluir eventos, para remover campanhas registradas indevidamente ou canceladas.**

---

### Critérios de Aceitação

| ID | Critério de Aceite | Status |
| :--- | :--- | :---: |
| **CA01** | Apenas eventos que não possuam doações vinculadas ou inscrições ativas podem ser excluídos. | <span class="badge-complete">completo</span> |
| **CA02** | Para eventos com histórico, o sistema deve permitir apenas a "Inativação/Cancelamento", preservando o histórico de dados. | <span class="badge-complete">completo</span> |

### Construção & Acesso

#### Painel de Exclusão de Eventos

* **Link para o sistema real:** [Acessar Portal Entre Amigos](https://github.com/mdsreq-fga-unb/REQ-2026.1-T01-PortalEntreAmigos.git)
* **Fluxo de Acesso:**
    1. Acesse a aplicação e autentique-se utilizando uma conta com perfil de "Moderador".
    2. Navegue até o painel ou listagem de gerenciamento de eventos da ONG.
    3. Escolha a campanha desejada e acione o botão ou comando de exclusão.
    4. O sistema verificará se existem dependências: caso não existam vínculos, o modal de exclusão física permanente será exibido; se houver histórico, a interface restringirá a ação permitindo apenas a inativação/cancelamento lógico.
    5. Confirme a operação no modal sobreposto para concluir o expurgo ou atualização de status.
